# Welcome to Dashboard Backus

Dashboard Backus is a comprehensive yard management system designed to optimize truck operations at the Backus facility. The platform provides real-time visibility into truck movements, bay assignments, and operational metrics through an intuitive satellite map interface.

## Technology Stack

Dashboard Backus is built with modern web technologies:

- **Frontend**: React 19, TypeScript, Vite, Tailwind CSS
- **Database & Auth**: Supabase (PostgreSQL)
- **Automation**: n8n webhooks for external integrations
- **Deployment**: Vercel

## How It Works

<Steps>
  <Step title="Authentication">
    Users log in with their credentials. The system supports two roles: **Admin** (can modify time thresholds) and **Client** (read-only access to thresholds).
  </Step>
  <Step title="Queue Monitoring">
    Trucks appear in the bottom queue panel as they arrive. Each truck card shows its type, plate, and current wait time with color-coded alerts.
  </Step>
  <Step title="Bay Assignment">
    Drag truck cards from the queue to compatible bays on the satellite map. The system validates truck type, operation, and product compatibility.
  </Step>
  <Step title="Real-Time Tracking">
    Monitor operations through floating dashboard panels showing priority units, shift statistics, and average yard time.
  </Step>
  <Step title="Incident Handling">
    Register incidents for trucks experiencing delays. The system tracks up to 3 incidents per truck and excludes incident time from performance metrics.
  </Step>
</Steps>
